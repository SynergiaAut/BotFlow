# Brand Kit - Skylab Human Bot

Carpeta de acceso rapido para logos, iconos y emblemas usados en la plataforma.

## Assets

- `skylab-horizontal-transparent.png`: logo principal de Skylab Human Bot. Usar en landing, login y headers destacados.
- `skylab-vertical-transparent.png`: version vertical del logo Skylab.
- `skylab-symbol-transparent.png`: simbolo Skylab sin wordmark.
- `skylab-app-icon.png`: icono app con contenedor.
- `skylab-icon.ico`: icono Windows para accesos directos.
- `synergia-wordmark-transparent.png`: logo Synerg-IA Automation para firmas sutiles.
- `synergia-symbol-transparent.png`: simbolo Synerg-IA para footers o creditos compactos.

## Regla Principal

Skylab Human Bot es la marca protagonista. Synerg-IA Automation debe funcionar como sello de respaldo, con menor escala/opacidad.
